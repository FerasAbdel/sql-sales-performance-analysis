# Sales Performance Analysis

## Overview
This project analyzes retail sales data to evaluate overall sales
performance and profitability across regions, product categories,
customer segments, and discount levels. The analysis is designed to
answer practical business questions using SQL.

## Business Questions
- Which regions generate the highest sales and profit?
- Which product categories and sub-categories drive revenue?
- How does discounting impact profitability?
- Which customer segments contribute most to sales and profit?

## Dataset
The analysis uses a retail sales dataset containing information on
orders, products, customer segments, regions, pricing, discounts,
and profit.

## Tools & Techniques
- SQL (aggregation, grouping, filtering)
- Business-focused analytical reasoning
- Data exploration using a web-based SQL environment

## Key Findings
- Sales and profit vary significantly by region and category.
- High sales volume does not always correspond to high profitability.
- Higher discount levels are often associated with reduced or negative profit.
- Customer segments differ in both revenue contribution and profitability.

## Repository Structure
- `queries.sql` — SQL queries used in the analysis
- `insights.md` — Business insights derived from query results

## Next Steps
Future extensions could include time-based analysis, visualization,
or deeper customer-level metrics if additional data is available.
